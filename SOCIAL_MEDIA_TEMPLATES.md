## Design Specifications Summary

### Color Usage Guidelines

| Platform | Primary Color | Background | Text |
|----------|---------------|------------|------|
| Instagram | Primary Blue + Bright Cyan | White / Light Gray | Deep Navy |
| Pinterest | Primary Blue gradient | White | Deep Navy / Slate Gray |
| X (Twitter) | Primary Blue / Deep Navy | White / Navy | Deep Navy / White |
| LinkedIn | Primary Blue accents | White | Deep Navy / Slate Gray |

### Typography Hierarchy

| Element | Font Weight | Size Range | Line Height |
|---------|-------------|------------|-------------|
| Headlines | Bold 700 / Extra Bold 800 | 32-120px | 1.2 |
| Subheadings | Semi-Bold 600 | 20-32px | 1.3 |
| Body Text | Regular 400 / Medium 500 | 18-24px | 1.4-1.5 |
| Labels | Medium 500 / Semi-Bold 600 | 16-18px | 1.2 |

### Spacing Standards

- Edge Padding: 40-80px depending on canvas size
- Element Spacing: 24-40px between major elements
- Line Spacing: 8-16px between related lines
- Logo Clearance: Minimum 40px from edges

### File Naming Convention

```
RDAI_[Platform]_[TemplateName]_[Dimensions].[Format]
```

Examples:
- `RDAI_Instagram_QuoteCard_1080x1080.png`
- `RDAI_LinkedIn_ProfessionalInsight_1200x627.png`
- `RDAI_Pinterest_ListPost_1000x1500.png`

## Content Adaptation Guide

### How to Customize Each Template

1. **Replace placeholder text** with actual content while respecting character limits
2. **Adjust layout** if content is shorter/longer (maintain spacing ratios)
3. **Choose appropriate color scheme** based on content tone:
   - Bright scheme: White/Light Gray bg for approachable content
   - Bold scheme: Navy/Blue gradient for professional/serious topics
   - Accent scheme: Cyan highlights for exciting/new announcements
4. **Add relevant icons** from approved icon set (24px grid, 2px stroke)
5. **Include custom images** following image treatment guidelines in BRAND_KIT.md

### Platform-Specific Best Practices

**Instagram**:
- Keep text minimal and highly readable
- Use vibrant colors and clear contrast
- Include engaging visuals or patterns
- Optimize for mobile viewing

**Pinterest**:
- Make headlines bold and scannable
- Use vertical space effectively
- Include clear value proposition
- Add "Pin it" or "Save" encouragement

**X (Twitter)**:
- Ensure text is readable at small preview size
- Use high contrast
- Keep headlines punchy and direct
- Include branding clearly

**LinkedIn**:
- Professional tone in design
- Clear, business-focused messaging
- Data and insights prominently featured
- Clean, uncluttered layouts
