## 1. Logo Concept

### Primary Logo Concept: "AI Wave"

**Visual Description**:
A modern, geometric logo combining a stylized "R" lettermark with a forward-moving wave pattern representing daily progress and AI data flow.

**Components**:
- **Lettermark "R"**: Constructed from geometric shapes (square + circle segment)
- **Wave Element**: 3-5 parallel curved lines flowing through/from the R, suggesting forward motion and data streams
- **Style**: Clean, minimal, tech-forward with precise geometry

**Construction Grid**:
- Base unit: 8px grid system
- Proportions: 1:1 ratio for square lockup, 16:9 for horizontal
- Clear space: Minimum 2x the height of the "R" on all sides

**Variations Needed**:
1. **Full Logo**: Lettermark + wordmark "RunDailyAI"
2. **Icon Only**: Lettermark + wave (for social avatars, favicons)
3. **Wordmark Only**: Text-only version for narrow spaces
4. **Monochrome**: Single-color versions (navy, white, black)

**Color Applications**:
- **Primary**: Deep Navy (#0A1628) on white/light backgrounds
- **Reversed**: White (#FFFFFF) on navy/dark backgrounds
- **Accent**: Primary Blue (#0066FF) for the wave element in digital applications
- **Gradient Option**: Subtle gradient from Primary Blue to Bright Cyan for premium digital contexts

**Typography in Logo**:
- Font: Inter ExtraBold (800)
- Letter-spacing: -0.02em (tight, modern)
- Case: Sentence case "RunDailyAI" or all caps "RUNDAILYAI" for impact

## 3. Typography Specifications

### Font Family
**Primary Font**: **Inter** (Google Fonts)
**Fallback**: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif

**Weights Used**:
- 400 (Regular) - Body text
- 500 (Medium) - Emphasis, labels
- 600 (Semi-Bold) - Subheadings, buttons
- 700 (Bold) - Headings
- 800 (Extra Bold) - Hero text, logo

### Type Scale

| Element | Desktop | Mobile | Weight | Usage |
|---------|---------|--------|--------|-------|
| **Hero/H1** | 48px/56px | 36px/44px | 800 (Extra Bold) | Landing page headers |
| **H2** | 36px/44px | 28px/36px | 700 (Bold) | Section headers |
| **H3** | 28px/36px | 24px/32px | 700 (Bold) | Subsection headers |
| **H4** | 24px/32px | 20px/28px | 600 (Semi-Bold) | Card headers |
| **H5** | 20px/28px | 18px/26px | 600 (Semi-Bold) | Small headers |
| **Body Large** | 18px/28px | 16px/24px | 400 (Regular) | Intro paragraphs |
| **Body** | 16px/24px | 16px/24px | 400 (Regular) | Standard text |
| **Body Small** | 14px/20px | 14px/20px | 400 (Regular) | Secondary text |
| **Caption** | 12px/16px | 12px/16px | 500 (Medium) | Labels, metadata |
| **Button** | 16px/24px | 16px/24px | 600 (Semi-Bold) | CTAs |

### Typography Guidelines

- **Line Height**: 1.5–1.75 for body text, 1.2 for headings
- **Line Length**: 60–80 characters maximum for readability
- **Letter Spacing**: -0.02em for headings, default for body
- **Paragraph Spacing**: 1.5em between paragraphs
- **Case**: Sentence case for most headings (not Title Case)

## 4. Iconography

### Icon Style
- **Style**: Outline (2px stroke) with optional solid fills for primary icons
- **Corner Radius**: 2px rounded corners for modern feel
- **Grid**: 24x24px base, scalable to 16px, 32px, 48px
- **Stroke Weight**: 2px (consistent across all icons)
- **Color**: Slate Gray (#64748B) default, Primary Blue (#0066FF) for active states

### Icon Sets
- **Recommended**: Heroicons, Lucide, Phosphor Icons (all match our outline style)
- **Custom Icons**: Follow same 2px stroke weight, 2px corner radius

## 10. Animation & Motion

### Timing Functions
- **Ease Out**: Default for UI interactions (cubic-bezier(0, 0, 0.2, 1))
- **Ease In Out**: For complex animations (cubic-bezier(0.4, 0, 0.2, 1))

### Duration Standards
- **Fast**: 150ms - Micro-interactions (hover, focus)
- **Normal**: 300ms - Standard transitions (page elements)
- **Slow**: 500ms - Complex animations (modals, drawers)

### Motion Principles
- **Subtle**: Prefer subtle animations over dramatic effects
- **Purposeful**: Every animation should communicate something
- **Performance**: Use transform and opacity for smooth 60fps animations
