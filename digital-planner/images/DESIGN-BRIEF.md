# Digital Planner Product Images - Design Brief

## Product Overview
**Product Name:** Digital Planner Bundle
**Format:** 5 CSV templates for Google Sheets/Excel
**Target Audience:** Professionals, students, solopreneurs
**Brand:** RunDailyAI (AI-focused, modern, clean aesthetic)

## Templates Included
1. Daily Planner - Time-blocking and task management
2. Weekly Planner - Week-at-a-glance view
3. Monthly Planner - Month overview with goals
4. Habit Tracker - Daily habit monitoring
5. Goal Tracker - Goal setting and progress tracking

## Design Guidelines

### Color Palette
- **Primary:** Modern blues, teals (tech/productivity feel)
- **Accents:** Energetic colors for highlights (green for completion, amber for focus)
- **Neutrals:** Clean whites, soft grays for backgrounds
- **Avoid:** Overly bright or clashing colors

### Typography
- **Headings:** Modern sans-serif (Inter, Roboto, SF Pro)
- **Body:** Readable sans-serif for any descriptive text
- **UI Elements:** Match Google Sheets/Excel interface style

### Composition
- Clean, uncluttered layouts
- Generous white space
- Clear visual hierarchy
- Professional mockups (if using device frames)
- Realistic sample data in templates (not lorem ipsum)

### Technical Specs
- **Format:** JPG (photographs/realistic mockups) or PNG (graphics with transparency)
- **Resolution:** High quality, web-optimized (72-150 DPI)
- **Color Space:** sRGB for web display
- **File Size:** Optimized for web (< 500KB each)

## Tools & Resources

### Recommended Design Tools
- **Canva:** Easy-to-use, templates available
- **Figma:** Professional, collaborative
- **Photoshop:** Advanced editing and mockups
- **Placeit/Smartmockups:** Device mockup generators

### Screenshot Approach
1. Populate each CSV template with realistic sample data
2. Open in Google Sheets with clean UI
3. Set appropriate zoom level (100-125%)
4. Take high-resolution screenshots
5. Clean up in photo editor
6. Add device mockups or composition elements
7. Export at required dimensions

### Quick Start Option
For fastest turnaround:
1. Use Canva's "Product Mockup" templates
2. Upload CSV template screenshots
3. Use pre-made device mockups
4. Customize text and branding
5. Export at specified dimensions
