## Technical Questions

### What format are the planners in?

All planners are delivered as **CSV (Comma-Separated Values)** files. This format works with:
- Microsoft Excel
- Google Sheets
- Apple Numbers
- LibreOffice Calc
- Notion (via CSV import)
- Airtable
- Any spreadsheet application

### Do I need special software?

No! CSV files work with any spreadsheet application. You can use:
- **Free options**: Google Sheets, LibreOffice
- **Paid options**: Microsoft Excel, Apple Numbers
- **Cloud tools**: Notion, Airtable, Coda

### Can I edit and customize the planners?

**Absolutely!** The CSV format is fully editable. You can:
- Add or remove columns
- Customize categories and fields
- Change formatting and styling
- Create your own formulas
- Duplicate sheets for multiple uses
- Merge data across planners

### What devices can I use these on?

The planners work on:
- **Desktop**: Windows, Mac, Linux
- **Mobile**: iOS (Numbers), Android (Google Sheets)
- **Web**: Any browser with Google Sheets or Excel Online
- **Tablet**: iPad, Android tablets

### How do I import into Notion?

1. Open Notion and create a new database
2. Click "Import" in the database menu
3. Select "CSV" as the file type
4. Upload the planner CSV file
5. Notion will create a database with all your planner fields

You can also use Notion's CSV import for inline tables.
