# Digital Planner

A comprehensive digital planning system with multiple spreadsheet templates.

## Files Included

1. **daily-planner.csv** - Daily planning template with time blocks, priorities, and reflections
2. **weekly-planner.csv** - Weekly overview with goals and priorities for each day
3. **monthly-planner.csv** - Monthly calendar with goal tracking
4. **habit-tracker.csv** - 31-day habit tracking grid
5. **goal-tracker.csv** - Quarterly and annual goal planning with milestones

## How to Use

### Google Sheets
1. Open Google Sheets
2. File → Import → Upload
3. Select a CSV file
4. Choose "Replace spreadsheet" or "Insert new sheet(s)"

### Microsoft Excel
1. Open Excel
2. File → Open
3. Select a CSV file
4. Use "Text Import Wizard" if needed

### Combining Into One Workbook
Import each CSV file as a separate sheet/tab in your spreadsheet application for a complete digital planner workbook.

## Tips

- **Daily Planner**: Print or use digitally for daily task management
- **Weekly Planner**: Review on Sundays for the week ahead
- **Monthly Planner**: Set at the start of each month
- **Habit Tracker**: Mark with X or ✓ for each completed day
- **Goal Tracker**: Review quarterly and update progress
