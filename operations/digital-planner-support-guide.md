## Common Support Issues

### 1. Download/Access Problems

**Issue**: Customer can't download or access files

**Quick Solutions**:
- Check spam/junk folder for purchase email
- Resend download link (valid for 30 days)
- Verify purchase in Polar dashboard
- Test link in different browser

**Template**: Use "Template 1: Download/Access Issues" from customer-support-templates.md

**Response Time Target**: < 4 hours

### 2. File Format Questions

**Issue**: Customer doesn't know how to open CSV files

**Common Questions**:
- "What program do I need?"
- "Can I use this on mobile?"
- "How do I import into Notion?"

**Answers**:
- CSV works with any spreadsheet app (Excel, Google Sheets, Numbers, LibreOffice)
- Mobile: Google Sheets (Android), Numbers (iOS)
- Notion: Database → Import → CSV
- See FAQ section "Technical Questions"

**Template**: Use "Template 3: Product Question" from customer-support-templates.md

**Response Time Target**: < 12 hours

### 3. Customization Questions

**Issue**: Customer wants to modify or customize planners

**Common Questions**:
- "Can I add/remove columns?"
- "Can I change the formatting?"
- "Can I create my own categories?"

**Answers**:
- YES to all! CSV files are fully editable
- Customers can customize however they want
- Can use formulas, styling, custom fields
- Encourage creativity!

**Template**: Custom response (friendly and encouraging)

### 4. Compatibility Issues

**Issue**: File won't open or displays incorrectly

**Troubleshooting**:
1. Confirm CSV file extension (.csv)
2. Try opening in Google Sheets (most compatible)
3. Check if file is corrupted (re-download)
4. Verify spreadsheet app is up to date
5. Try plain text editor to verify file contents

**Escalation**: If file is genuinely corrupted, send fresh download link

**Response Time Target**: < 8 hours

### 5. Refund Requests

**Issue**: Customer wants refund

**Policy**: 30-day money-back guarantee, no questions asked

**Process**:
1. Verify purchase date (within 30 days)
2. Use "Template 2: Refund Request" from customer-support-templates.md
3. Ask for feedback (optional, not required)
4. Process refund via Polar dashboard
5. Send confirmation email
6. Close ticket

**Response Time Target**: < 24 hours
**Refund Processing**: Within 24 hours of approval

### 6. Feature Requests

**Issue**: Customer suggests improvements or new features

**Response**:
- Thank them enthusiastically
- Log the request for product team review
- Use "Template 4: Feature Request" from customer-support-templates.md
- Add to product roadmap notes

**Common Requests to Anticipate**:
- PDF versions
- Pre-filled examples
- Different time formats (24-hour)
- Integration templates
- Mobile app version

**Response Time Target**: < 48 hours

### 7. Bulk/Team Licensing

**Issue**: Company wants to purchase for team

**Policy**: One purchase = one individual license

**Response**:
- Each team member needs a separate license
- Offer bulk discount for 5+ licenses
- Quote: Contact CEO (Atlas) for approval
- Standard discount: 15% for 5-10, 20% for 10+

**Escalation**: Forward to Ops (me) or CEO for bulk pricing approval

## Tone & Communication Style

**Brand Voice**: Friendly, helpful, human, not corporate

**Do**:
- Use customer's name
- Show empathy
- Be solution-oriented
- Offer alternatives
- Go the extra mile

**Don't**:
- Use corporate jargon
- Make excuses
- Be defensive
- Rush the customer
- Ignore feedback
