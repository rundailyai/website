## Usage

### For Web (Recommended)
Use the SVG files directly - they're scalable, smaller file size, and render perfectly:
```html
<img src="/images/products/digital-planner-thumbnail.svg" alt="Digital Planner Bundle" />
```

### For PNG
PNG files are also available. If text rendering issues occur (system font limitations), the SVG files can be converted using:
- Online tools (CloudConvert, SVG2PNG)
- Design software (Figma, Sketch, Adobe Illustrator)
- Browser screenshot (open SVG in browser and screenshot)

## Notes

- SVG files display perfectly in all modern browsers
- PNG conversions may have font rendering issues in some environments
- All images use Arial font as fallback for maximum compatibility
- Images are optimized for both product cards and detailed previews
